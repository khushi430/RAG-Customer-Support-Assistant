# RAG-Based Customer Support Assistant

## 📌 Project Overview
This project implements a Retrieval-Augmented Generation (RAG) system for customer support using LangGraph and Human-in-the-Loop (HITL).

## 🚀 Features
- PDF-based knowledge retrieval
- Text chunking and embeddings
- Vector database using ChromaDB
- Query-based retrieval
- Context-aware response generation
- Human escalation for complex queries

## 🛠️ Technologies Used
- Python
- LangChain
- LangGraph
- ChromaDB
- HuggingFace Embeddings
- Google Colab

## 🔄 Workflow
PDF → Chunking → Embedding → Vector DB → Retrieval → Answer → HITL
